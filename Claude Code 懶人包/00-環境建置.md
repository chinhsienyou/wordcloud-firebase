# 00-環境建置

Claude Code 的基礎開發環境設定。

## 必要工具

- Node.js 16+ 或 Python 3.8+
- Git
- 文字編輯器（VS Code、Cursor 等）

## 安裝步驟

### 1. 安裝 Node.js（可選）

```bash
# macOS with Homebrew
brew install node

# 驗證安裝
node --version
npm --version
```

### 2. 安裝 Git

```bash
# macOS
brew install git

# 驗證安裝
git --version
```

### 3. 設定 Git 用戶名

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

## 常見問題

### Q: 如何檢查已安裝的工具版本？

```bash
node --version
npm --version
git --version
```

### Q: 我應該選擇哪個編輯器？

推薦使用 VS Code 或 Cursor，並搭配 Claude Code 擴充功能。
